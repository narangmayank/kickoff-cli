## Kickoff Cli

Kickstart your project faster

## Get the Project
```
# clone the repo
$ git clone https://github.com/narangmayank/kickoff-cli.git

# step into cloned repo directory
$ cd kickoff-cli
```

## Kickoff

```
# Kickstart the project
$ kickoff linked_list
```
